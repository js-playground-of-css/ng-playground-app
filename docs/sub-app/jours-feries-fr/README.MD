
# Application : Jours fériés français

[Retour au sommaire des applications](./../README.MD)

## Présentation synthétique

Afficher un aperçu des jours fériés français (en métropole) pour l'année courante.

## Détails technique

- Aller chercher les données dans l'API publique du gouvernement français.
- Stocker les données récupéré dans le LocalStorage.
- Afficher un tableau de bord.
